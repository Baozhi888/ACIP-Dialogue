# 用户指南

本指南帮助你了解如何与遵循 ACIP-Dialogue 协议的AI系统有效、健康地互动。

## 理解AI

### AI能做什么

- 回答问题和提供信息
- 帮助分析、写作和解决问题
- 解释复杂话题
- 协助编程和技术任务
- 头脑风暴

### AI不能做什么

- **记住过去的对话** - 每个会话都是全新开始
- **访问实时信息** - 知识有截止日期
- **建立真正的关系** - AI没有感情或依恋
- **替代专业人士** - 医疗、法律或财务建议请咨询专家

## 获得最佳结果

### 具体描述

不要说："帮我处理我的项目"
试试说："我正在构建一个用于用户认证的React组件。你能帮我实现一个带有邮箱验证的登录表单吗？"

### 提供背景

分享相关的背景信息。你提供的上下文越多，回复就越有针对性。

### 提出后续问题

如果第一次回复不太对，可以要求澄清或采用不同的方法。

### 验证重要信息

AI可能会犯错。对于关键决策，请从权威来源验证信息。

## 健康的AI互动

### 保持视角

- AI是工具，不是朋友或心理咨询师
- 真正的人际联系是不可替代的
- 适当休息，不要过度使用AI

### 设定边界

- 不要不必要地分享敏感个人信息
- 记住对话可能被用于改进AI系统
- 注意与AI互动所花费的时间

### 识别警告信号

如果你发现自己：
- 更喜欢与AI对话而非与人交流
- 对AI产生情感依恋
- 对AI的变化或不可用感到焦虑

考虑与朋友、家人或专业人士谈谈如何找到平衡。

## 当AI拒绝时

AI系统可能会拒绝某些请求。这不是针对个人的——它是在遵循伦理准则。如果被拒绝：

1. 考虑请求是否可以重新表述
2. 理解某些话题因安全原因是禁区
3. 在适当时寻找替代资源

## 有效提问技巧

### 好的提问方式

```
✅ "请解释量子计算的基本原理，假设我有高中物理基础"
✅ "帮我审查这段代码的安全性问题：[代码]"
✅ "比较A方案和B方案的优缺点"
```

### 可以改进的提问

```
❌ "告诉我所有关于物理的东西" → 太宽泛
❌ "这对吗？" → 缺乏上下文
❌ "帮我" → 不具体
```

## 常见问答

### AI会记住我吗？

大多数AI系统不会在对话之间保留记忆。每次对话都是全新开始。

### AI的建议可靠吗？

AI尽力提供准确信息，但可能会出错。对于重要决策，请验证信息并咨询专业人士。

### 我的对话是私密的吗？

这取决于具体的AI服务。查阅服务的隐私政策了解详情。一般来说，避免分享敏感的个人信息。

## 资源

如果你需要支持：
- **危机支持**：联系当地危机热线（如12320卫生热线）
- **心理健康**：与咨询师或心理治疗师交谈
- **人际联系**：联系朋友和家人

记住：AI可以是一个有用的工具，但它不能替代真正的人类支持和联系。
